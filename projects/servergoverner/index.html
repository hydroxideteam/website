What is this?

ServerGoverner is a plugin for the Bukkit Minecraft server, built to allow server administrators more control and security over their server, while maintaining the "Vanilla" server feel that other similar plugins disregard.

Why do I need it?

ServerGoverner is a lightweight and simple administration tool for your server, with enhanced banning and kicking, as well as adding the ability to mute and jail rule-breakers on your server.

Download and Installation

Please download the plugin from our BukkitDev page.

First of all, make sure your Minecraft server is running Bukkit or Spigot.

Download the appropriate version for your server, eg v1.7.2 .

If you cannot find your version, you can try a nearby

Next, drag the downloaded JAR file to the 'plugins' folder, inside your server folder.

That's it! Just restart or reload your server, and the plugin should be usable

Commands and Permissions

Command Permission Node Description
/local [message] sgov.localchat Sends a message to nearby players within radius.
/home sgov.home Teleports the player to the last bed.
/quit sgov.commandquit Allows the player to leave the server via command.
/data sgov.data Gives a log of data to the player. Location, health, etc.
/god sgov.god Gods the player
/lag sgov.lag Displays server health stats such as RAM usage.
/friend [add|remove] [player] or /friend list or /friend c [message] sgov.friend Friend system command.
/fc [message] sgov.friend Shortening of friend chat command.
/sg sgov.admin ServerGoverner Admin commands
/ban [player] [optional time] [reason] sgov.admin.ban Allows banishment of other players.
/kick [player] [optional reason] sgov.admin.kick Allows jailing of other players.
/mute [player] [time] sgov.console.mute Allows muting of other players.
/jail [player] [time] sgov.console.jail Allows jailing

Notes:
You can specify whether you want days, hours, or minutes in [time] tags, ie: 5m, 12h
